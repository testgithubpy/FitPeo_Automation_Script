# FitPeo Revenue Calculator Automation Script

## Description
This script automates the process of navigating to the FitPeo Revenue Calculator page, adjusting the slider, and selecting checkboxes using Selenium WebDriver.

## Setup Instructions

### Prerequisites
1. Python 3.x installed on your system.
2. Install Selenium WebDriver:

pip install selenium
3. Download the appropriate WebDriver (e.g., [ChromeDriver](https://sites.google.com/a/chromium.org/chromedriver/downloads)) and ensure it's accessible in your PATH.

### Running the Script
1. Place the script (`main.py`) and the WebDriver executable in the same directory.
2. Open a terminal or command prompt.
3. Navigate to the directory containing the script.
4. Run the script using:

python main.py

## Important Notes
- Ensure the WebDriver version matches your browser version.
- Adjust the path to the WebDriver executable in the script if necessary.
- The script includes exception handling to manage potential errors during execution.
